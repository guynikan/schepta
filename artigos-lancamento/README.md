# Artigos de lançamento do Schepta

Série de cinco artigos em português para o lançamento do Schepta. Podem ser publicados em blog, Medium, dev.to ou no site do Schepta.

## Ordem da série

1. **[Por que o Schepta existe](01-por-que-o-schepta-existe.md)** — O que é e que problema resolve (não técnico); que espaço o Schepta preenche.
2. **[Por que não fui pelo caminho comum](02-por-que-nao-fui-pelo-caminho-comum.md)** — Análise do momento atual no desenvolvimento de software; microsaas e agentes de IA; por que lib open source (e não microsaas) e por que não (ainda) apostar em agentes que resolvem tarefas.
3. **[Schepta no mundo das IAs](03-schepta-no-mundo-das-ias.md)** — Validação (json-render, A2UI), não-determinismo das IAs, fluxos do código assistido ao 100% automatizado.
4. **[Onde e como usar o Schepta](04-onde-e-como-usar-o-schepta.md)** — Planos futuros, outros usos (multi-tenant, A/B, wizards, white-label), testes como garantia, informações práticas.
5. **[Como o Schepta funciona por dentro](05-como-o-schepta-funciona-por-dentro.md)** — Factories, orquestradores, registry, pipeline e referências ao código.

Cada artigo termina com link para o próximo e para [schepta.org](https://schepta.org).
