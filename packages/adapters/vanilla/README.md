# @schepta/adapter-vanilla

Vanilla JavaScript adapter for schepta. Provides a framework-agnostic runtime adapter for use without React or Vue.

## Installation

```bash
npm install @schepta/adapter-vanilla
```

## Usage

```typescript
import { VanillaRuntimeAdapter } from '@schepta/adapter-vanilla';

const runtime = new VanillaRuntimeAdapter();

// Use with FormFactory
```
